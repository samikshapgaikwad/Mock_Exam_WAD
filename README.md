# 🍽️ Delicious Bites – Restaurant Web App

## 📌 Overview

Delicious Bites is a simple Node.js-based restaurant web application that provides:

* A home page with restaurant details
* A menu page displaying food items with pricing
* Static frontend using HTML and CSS

---

## 🚀 Features

* 🏠 Home page with About and Contact sections
* 📜 Menu page with categorized items:

  * Starters
  * Main Course
  * Desserts
* 🎨 Styled UI using CSS
* 🌐 Basic routing using Node.js server

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Node.js (Express)
* **Environment:** Localhost server

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/samikshapgaikwad/Mock_Exam_WAD.git
```

2. Navigate to the project folder:

```
cd Mock_Exam_WAD
```

3. Install dependencies:

```
npm install
```

4. Run the server:

```
node server.js
```

5. Open in browser:

```
http://localhost:3000
```
